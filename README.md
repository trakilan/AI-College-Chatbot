🎓 AI College Chatbot

An AI-powered College Information Chatbot developed for **Thiagarajar College of Engineering (TCE)**. The chatbot answers student queries related to admissions, placements, contact information, and other college-related topics using **Google Gemini AI**, **FastAPI**, and **ChromaDB**.

---

🚀 Features

- 🤖 AI-powered chatbot using Google Gemini
- 📚 Retrieval-Augmented Generation (RAG)
- 🔍 Semantic search using ChromaDB
- ⚡ FastAPI backend
- 💻 Simple HTML, CSS & JavaScript frontend
- 📄 College information stored in text documents
- 🧠 Context-aware responses
- 🌐 REST API support

---

🛠️ Technologies Used

- Python 3.12
- FastAPI
- Google Gemini AI
- ChromaDB
- Uvicorn
- HTML
- CSS
